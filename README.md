# Yoshi's zones

## Correr Localmente

Clona el proyecto:

```bash
git clone https://github.com/SARTDv/yoshis_zonesDocker.git
```

Entra al directorio del proyecto:

```bash
cd yoshis_zonesDocker
```

Inicia la app con:

```bash
bash ./iniciar.sh
```

Ahora solo debes entrar en [Localhost](localhost:5173)
